# 2. Controls


```
The controls for Phantasy Star IV are very simple, requiring only a three-
button control pad. They are as follows:


Title Menu:

   START - Select option
   A,C   - Select option
   B     - Cancel choice, open Sound Test (once game is completed)
   D-Pad - Scroll options


The Field:

   START - Open System Menu
   A     - Open Party Menu
   B     - Cancel choice
   C     - Talk, open, use, read, examine, select option
   D-Pad - Walk in that direction


Game/Party Menu:

   START - Exit Menu
   A,C   - Select option
   B     - Cancel choice
   D-Pad - Scroll options


Battle:

   START - No function
   A,C   - Select option
   B     - Cancel choice
   D-Pad - Scroll options

```
